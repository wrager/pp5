# Лабораторная №7. Параллельное программирование на C#.

Переделать приложение лабораторной работы №6 на язык C#.


## Требования по процессу

1. Работа выполняется в форке этого репозитория.

2. Код должен находиться в отдельной ветке в директории `/lw7/<номер варианта>/<Фамилия>_<Имя>/`.

3. После завершения работы сделать pull request вашей ветки в ветку `lw7` этого репозитория с заголовком `<Фамилия> <Имя> lw7 (<номер варианта>)`.

Никаких лишних изменений в pull request'е быть не должно - только добавление файлов в папку `/lw7/<номер варианта>/<Фамилия>_<Имя>/`!
